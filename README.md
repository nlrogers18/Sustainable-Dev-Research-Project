# AI-Research-Project

- Contains exploratory data analysis, modeling experiments, and data visualizations across multiple CSV and notebook files.
- Includes the Jupyter notebook, a compressed archive of the full project, and a mock IEEE research paper used as reference material.

## Root Directory

- **`AI_Project.ipynb`**  
  Main notebook containing preprocessing, exploratory analysis, model experimentation, and results.

- **`AI_Research_Project.zip`**  
  Compressed archive of the full project structure for portability and backup.

- **`IEEE Paper.pdf`**  
  Research paper used as a theoretical reference for the presentation of analysis and predictive modeling.

- **`renewable.csv`**  
  Dataset containing renewable‑energy‑related variables.

- **`temp.csv`**  
  Dataset containing Temperature variables over time.

## Requirements to Run

To run this project, you will need the following dependencies:

### Python Libraries

- **`pandas`** — Data manipulation and analysis  
- **`numpy`** — Numerical operations and array handling  
- **`scikit-learn`** — Machine learning algorithms, preprocessing utilities, and model evaluation tools
- **`scipy.stats`** — Statistical functions, probability distributions 
- **`matplotlib`** — Plotting library for creating static, animated, and interactive visualizations  
- **`seaborn`** — High‑level statistical data visualization built on top of Matplotlib, with cleaner styles and easier plotting
